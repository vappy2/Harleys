# Harleys #

### Description du projet ###
* Application permettant de crée une base de donnée (ici sous MySQL) via des classes C#.
* On utilise ici la méthode Code First

##### Cas d'utilisation ####
*inséré le CU*

#### Diagrame de classe ####
*insérer le DDC* 

#### Langage & outils ####
* Visual Studio 2017 (C#)
* MySQL 8.0
